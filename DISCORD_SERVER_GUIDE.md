## 📜 Server Rules

1. **Be Respectful** - Treat everyone with respect and kindness
2. **Stay On Topic** - Use appropriate channels for discussions
3. **No Spam** - No self-promotion or spam content
4. **English Primary** - Use English in main channels
5. **No Piracy** - No discussion of pirated software
6. **Follow Discord ToS** - Adhere to Discord's Terms of Service

## 📞 Support Tickets

Use the Ticket Tool bot to create support tickets:

1. React to the ticket message in `#support`
2. A private channel will be created
3. Describe your issue in detail
4. A support team member will assist
